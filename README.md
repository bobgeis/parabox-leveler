# parabox-leveler

This is a level editor for [Patrick's Parabox](https://store.steampowered.com/app/1260520/Patricks_Parabox/). It can be used to create custom levels and then download them as txt files. [Try it](https://bobgeis.github.io/parabox-leveler/).

Custom levels should be stored in
  - MacOS: /Users/(Username)/Library/Application Support/com.PatrickTraynor.PatricksParabox/custom_levels
  - Windows: C:\Users\(Username)\AppData\LocalLow\Patrick Traynor\Patrick's Parabox\custom_levels
